# 🧠 Brain Tumor Detection

A deep learning project that detects brain tumors from MRI scans.

## Algorithms
- MobileNetV2 — 88.4% accuracy
- CNN — 78.5% accuracy  
- SVM — 88.7% accuracy

## Datasets
- Brain Tumor MRI Dataset (Kaggle)
- Brain Tumor Classification MRI (Kaggle)

## Usage
1. Install requirements: `pip install -r requirements.txt`
2. Train models: `py train.py`
3. Run app: `py app.py`
